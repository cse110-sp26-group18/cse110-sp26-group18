# Ideas & Brainstorm

## Core Features
- Upload an image of someone or something and use a prompt to change the image with AI
- Use AI to change facial expression / features on uploaded images
- Meme library with AI-suggested captions
- Custom fonts and labels with a daily usage limit
- Use AI to finish meme generation tasks started by the user (e.g. generate a caption)
- Use AI to change existing captions of an uploaded meme
- AI should be able to create motion in memes
- Low quality aesthetic adds charm — lean into it
- Search engine functionality, similar to GIPHY
- Start with photos, then explore video support

## References & Inspiration
- ImgFlip — reviewed their AI feature, found it lacking
- TikTok stickers, face swap features
- GIPHY
- iMessage stickers (but with captions)

## Distribution & Saving
- Initially no login required — save memes locally
- App should send the generated meme directly to messages
- Regular download feature should also be included
- PWA (web app with option to add to iPhone/Android home screen)

## Testing Groups
- Divide into mini sprint groups for initial testing: survey, face swapping, meme generation, etc.
- Run mini sprints at the end of class on Tuesdays and Thursdays
